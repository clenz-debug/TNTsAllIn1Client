# Designs

Sammelstelle für alles Visuelle rund um Client + Launcher: Mockups, Referenz-Screenshots, Farbpaletten, Icons/Logo. Kein Code — reine Design-Assets und Notizen dazu.

## Struktur

```
designs/
├── launcher/       Screens/Mockups für den Electron-Launcher (Login, Versionsauswahl, Mods, Skin-Editor, Freunde)
├── ingame/         Ingame-Menü, HUD-Elemente (Koordinaten/Kompass, Material-Zähler, F3-Erweiterungen)
├── branding/       Logo, Icon, Farbpalette, Typografie — projektweit einheitlich
└── inspiration/    Referenz-Screenshots anderer Clients/Launcher (Lunar, Badlion, Feather, Dawn) als Vorbild, nicht zum 1:1-Kopieren
```

Format ist offen — PNG/JPG-Screenshots, exportierte Figma-Frames, oder auch nur eine `notes.md` mit Beschreibung/Links, wenn (noch) kein fertiges Bild existiert. Dateinamen wie in den restlichen `.md`-Dateien des Projekts: Deutsch, `Groß_Mit_Unterstrichen` oder sprechende_kleinschreibung, Hauptsache konsistent innerhalb eines Unterordners.
